completer(line) → {Array.<String>}

The completion callback function, this method is invoked everytime the user presses the tab key. Such implementation allows for easier testing as well as extension through replacing this method with an own implementation. The method returns an array of strings containing two elements, the first element being the possible completions for the user input as an array of string, the second is what the prompt should be. 

    return [['test','test-ng','test-task'],'te'];
